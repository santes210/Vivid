# Vivid - Instagram-like App (Material You 3 + Jetpack Compose)

**Nombre:** Vivid  
**Icono:** Personalizado (gradiente moderno)

## ✅ Estado final - Todo implementado

### Características completas

**Autenticación**
- Email/contraseña + Google (preparado)
- Sesión persistente

**Feed + Historias**
- Scroll infinito
- Stories con visor completo (K)
- Avatares cacheados (Coil)

**Crear contenido**
- Publicar posts (cámara + galería + Firebase Storage)
- Reels con ExoPlayer (E)
- Subida real a Storage

**Mensajes (DM)**
- Tiempo real con Firestore
- Notificaciones push que abren el chat (H)

**Búsqueda y Social**
- Búsqueda de usuarios (G)
- Seguir / Dejar de seguir (I) - FollowRepository
- Contadores de seguidores

**Perfil**
- Editar perfil completo (L)
- Subir foto de perfil
- Grid de publicaciones

**Reels**
- Reproducción vertical automática
- Posibilidad de subir reels reales (J)

**Técnico**
- Room (caché offline)
- Hilt + MVVM
- Material You 3 + Dynamic Colors
- GitHub Actions (compilación automática)

---

## Cómo compilar

### Opción 1: Android Studio
1. Abre el proyecto
2. Sincroniza
3. Run

### Opción 2: GitHub Actions (sin PC)
1. Sube a GitHub
2. Ve a **Actions** → Run workflow
3. Descarga el APK
