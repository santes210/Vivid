## Cómo usar

### Opción 1: Instalar el APK directamente
1. Copia `apk/app-debug.apk` a tu teléfono
2. Instálalo (activa "orígenes desconocidos")
3. Backblaze ya está configurado con tus credenciales

### Opción 2: Compilar desde Android Studio / GitHub Actions
1. Descomprime este ZIP
2. Abre la carpeta `vivid-app/` en Android Studio
3. Espera a que sincronice Gradle → Run

Para subir a tu repo de GitHub:
```bash
git init   # si es nuevo
git add .
git commit -m "Vivid con build + Backblaze B2 corregidos"
git push origin main
```
GitHub Actions compilará el APK automáticamente.
